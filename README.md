# word
word.md 中记录了每天或隔几天的较为难记的单词汇总
********************************************
## 翻译单词app(node) 0.1.0 版 (翻译功能基于有道翻译api)
# 使用方法
>在有道智云申请文本翻译应用

>将应用ID 应用密钥 填到 appKey secureKey中 保存

>安装node环境(版本控制在官网最新稳定版之上)

>在yourword.txt中逐行写下你的单词

>在当前文件夹下运行命令 node runThis.js

>在result.txt中查看翻译结果
